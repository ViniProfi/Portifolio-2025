Meu Primeiro Portfólio: Uma Página Pessoal no GitHub

Este projeto é uma página web desenvolvida com HTML e hospedada diretamente no GitHub Pages. É o meu primeiro portfólio online e foi criado com o objetivo de apresentar um pouco sobre mim, minhas curiosidades e minha jornada na programação. A proposta é simples, porém significativa: ter um espaço pessoal na internet que represente quem sou e mostre meu crescimento na área da tecnologia.

A estrutura do site foi construída utilizando HTML puro, com seções organizadas para facilitar a leitura e navegação. A página contém uma breve apresentação, incluindo minha idade, meus interesses, uma curiosidade pessoal e um meio de contato direto via e-mail. Também inclui ícones e elementos visuais como gifs, que foram adicionados para deixar a experiência mais divertida e expressiva.

Um dos principais desafios durante o desenvolvimento foi aprender a organizar o conteúdo e inserir gifs corretamente, o que exigiu pesquisa e prática. O processo de subir o projeto no GitHub e configurar o GitHub Pages foi um aprendizado importante, que me ajudou a entender melhor como funciona a publicação de sites na web. tela.

Apesar da simplicidade, este projeto representa uma grande conquista pessoal. Foi através dele que aprendi os fundamentos do HTML, explorei o uso da plataforma GitHub e comecei a entender os bastidores da criação de páginas na internet. O site ainda está em constante evolução, e representa o início de muitos projetos que virão no futuro.

Agrinho: Do campo à cidade, colhendo oportunidades.

Este projeto foi desenvolvido na plataforma Scratch com o objetivo de criar uma experiência interativa e educativa sobre o caminho dos alimentos, desde a colheita no campo até a chegada nas feiras e mercados da cidade. A ideia surgiu a partir do tema “Agrinho” e foi desenvolvida de forma totalmente original, com roteiro, personagens e interações criadas por mim.

O jogo acompanha a trajetória de um jovem agricultor que enfrenta desafios e toma decisões importantes para garantir uma boa colheita, proteger os produtos e levá-los com sucesso até o consumidor final. A narrativa é leve e envolvente, trazendo elementos como obstáculos, fases e recompensas, que tornam a experiência mais dinâmica e divertida.

O uso do Scratch permite uma programação por blocos acessível, mas cheia de possibilidades. Nesse projeto, utilizei recursos de movimentação, detecção de colisão, mudanças de cenário e efeitos visuais para construir o ambiente do jogo. Também trabalhei com lógica condicional e variáveis para controlar pontuação e progresso.

O principal objetivo do projeto é promover a conscientização sobre o valor do trabalho no campo e a importância da agricultura para a sociedade, mostrando de forma lúdica todo o esforço envolvido até os alimentos chegarem à cidade. Além disso, o jogo estimula a tomada de decisões e o pensamento estratégico.

Este foi um dos projetos que mais me empolgaram até agora, tanto pelo tema quanto pela liberdade criativa. Ele representa não só um exercício de programação, mas também uma forma de expressar ideias importantes de maneira acessível e divertida.
